Let’s talk about Javascript as language

  • A fact: Is the only language supported by web browsers that allow to write interactive web applications.

  • For small-scale is OK, the verbosity or the surprising behaviour are acceptable for few hundred lines of code-base.

  • However, is not an easy language to work when your code-base extends to thousands, tens or hundreds of thousands of lines of code due to the untyped nature of the language. It can be hard to test all code and consequently refactoring is far from trivial, becoming dangerous, indeed.

  • The big deal: The language has left many things unspecified, yet at the same time provides the ability to emulate them in several ways.
