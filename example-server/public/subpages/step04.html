What is Scala.js

In short: Scala.js allow compile your Scala code to a Javascript executable that can run on all major web browsers.

  • You get all the benefits of the web platform with none of the problems of writing your software in Javascript.

  • You get all the powerful Scala features like type inference, classes, traits and objects, pattern matching, the collections library, implicits, etc.

  • And the most notable feature never been seen in web development: shared-code between client & server.
