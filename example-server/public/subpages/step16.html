DOM interaction

To use the DOM, it is best to use the statically typed Scala.js DOM library. To add it to your sbt project, add the following line to your build.sbt:

libraryDependencies += "org.scala-js" %%% "scalajs-dom" % "0.8.0"

import org.scalajs.dom.document

def addParagraph(targetNode: dom.Node, text: String): Unit = {
    val parNode = document.createElement("p")
    val textNode = document.createTextNode(text)
    parNode.appendChild(textNode)
    targetNode.appendChild(parNode)
}

Now we can create a paragraph, since main method for instance:

def main(): Unit = {
    addParagraph(document.body, "Hello World")
}
