Pattern matching

Scala has a built-in general pattern matching mechanism. It allows to match on any sort of data with a first-match policy.

def matchTest(x: Int): String = x match {
    case 1 => "one"
    case 2 => "two"
    case _ => "many"
}

Pattern matching can wildcard parts of expressions:

def techs(expr: Any) = expr match {
    case ("Javi", _) => "Scala"
    case ("Israel", "Frontend") => "Javascript"
    case ("Rafa", "Frontend") => "Scala JS"
    case (_, "iOS") => "Swift"
    case _ => "Scala"
}
> techs("Javi", "Android") //Scala
> techs("Ana", "iOS") //Swift
