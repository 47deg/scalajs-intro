Pattern matching

Scala has a built-in general pattern matching mechanism. It allows to match on any sort of data with a first-match policy.

def matchTest(x: Int): String = x match {
    case 1 => "one"
    case 2 => "two"
    case _ => "many"
}

For instance, we can use pattern matching for creating the step function:

def stepFunc(x: Double): Double = x % 10 > 5 match {
    case true => 0
    case _ => 2
}

Can I create a new graph with stepFunc?

val stepGraph = Graph(2, Color("blue"), stepFunc)

With a new list of Graphs:

val graphs = List(zeroGraph, sinGraph, stepGraph)
