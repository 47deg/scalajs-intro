User interaction and interoperability with Javascript

Expose a Scala method, to be called from Javascript code is easy, you only have to add the annotation @JSExport over the method:

@JSExport
def addParagraph(targetNode: dom.Node, text: String): Unit = {
    val parNode = document.createElement("p")
    val textNode = document.createTextNode(text)
    parNode.appendChild(textNode)
    targetNode.appendChild(parNode)
}

This way we can bind a click event to fire the method:

<button type="button" onclick="demo.UserInteraction().addParagraph(document.body, 'Button pushed')">Click me!</button>
