Pattern matching

To pattern match against a List:

val secondElement = List(1, 2, 3) match {
    case x :: xs => xs.head
    case _ => 0
}
val secondElement = List(1,2,3) match {
    case x :: y :: xs => y
    case _ => 0
}
val secondElement = List(1) match {
    case x :: y :: xs => y
    case _ => 0
}
