Case classes

They are regular class with special privilegies. 

case class User(
    firstName: String,
    lastName: Option[String] = None,
    postalCode: String,
    pictures: Media)

case class Media(
    large: String,
    medium: String,
    thumbnail: String)

Case classes have a convenient way they can be created, new is not required

val rafaPics = Media("photo400.jpg", "photo200.jpg", "photo50.jpg")
val rafaUser = User("Rafa", Some("Paradela"), "11130", rafaPics)

One can simply use the class name as a function. And compose nested instantiations.

val fedeUser = User("Fede", None, "41130", Media("pic400.jpg", "pic200.jpg", "pic50.jpg"))
