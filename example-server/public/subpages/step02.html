Let’s talk about Javascript as platform

The platform has some very nice properties that make it a good target for application developers:

  • Zero-install distribution: just go to a URL and have the application downloaded and ready to use.
  • Hyperlinks: being able to link to a particular page or item within a web app is a feature other platforms lack.
  • Sandboxed security: you can be sure that once you close the page, they’re gone.
