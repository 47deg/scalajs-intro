What about jQuery?

In asdf build.sbt:

libraryDependencies += "be.doeraene" %%% "scalajs-jquery" % "0.8.0"
skip in packageJSDependencies := false

Include the generated javascript dependencies file:

<script type="text/javascript" src="./target/scala-2.11/demo-jsdeps.js"></script>

Now you can:

import org.scalajs.jquery.jQuery

object JqueryExample  {

    def main() = {
        jQuery("button").click((e: js.Any) => jQuery("body").append("<p>jQuery is working!</p>"))
    }

}
