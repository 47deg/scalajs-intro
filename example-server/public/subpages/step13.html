Case classes

They are regular class with special privilegies. 

case class Color (name: String)
case class Graph (id: Int, color: Color, func: Double => Double)

Case classes have a convenient way they can be created, new is not required

case class Color (name: String)
case class Graph (id: Int, color: Color, func: Double => Double)

One can simply use the class name as a function. And compose nested instantiations.

val sinGraph = Graph(1, Color("green"), x => sin(x))
